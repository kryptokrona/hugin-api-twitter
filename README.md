# Setup

To setup this project locally to test:

- git clone git@github.com:kryptokrona/hugin-api-twitter.git (or use HTTPS)
- cd hugin-cache-twitter
- virtualenv -p python3 .
- source bin/activate
- pip3 install -r requirements.txt
